# Reference folder

This folder contains **Component Reference** — a UI that documents all PageDef components, their props, and how to use them in JSON.

- **ComponentReference.tsx** — Renders the reference page.
- **component-reference-data.ts** — Data for component types, props, and example JSON.

You can delete this entire folder when you no longer need the reference. After removing it:

1. Delete the `reference` view and `ComponentReference` import from `App.tsx`.
2. Remove the `reference` folder.
